________________________________________
1️⃣ PROJECT OVERVIEW
The Restaurant POS (Point of Sale) System is a complete restaurant management solution designed to automate daily restaurant operations including:
•	Restaurant Authentication
•	Staff Management
•	Table Management
•	Menu & Category Management
•	Inventory Control
•	Order Processing
•	Kitchen Order Handling
•	Billing & Payments
The system is built using:
•	Backend: Node.js / Express
•	Frontend: React
•	Database: SQL Server (with schemas, triggers, constraints)
•	Role-Based Authentication
•	Real-time Kitchen Flow
•	Automated Billing Calculation
The system follows industry-level architecture using separate schemas:
•	auth → Authentication related tables
•	master → Restaurant master data
•	orders → Orders & Order Items
•	kitchen → Kitchen operations
•	billing → Bills & Payments

4️⃣ COMPLETE SYSTEM FLOW (STEP-BY-STEP)
________________________________________
🟢 STEP 1: Restaurant Login
•	Admin logs into system.
________________________________________
🟢 STEP 2: Setup (Initial Configuration)
Admin:
•	Adds Categories
•	Adds Menu Items
•	Adds Tables
•	Adds Staff
•	Adds Inventory items
________________________________________
🟢 STEP 3: Customer Comes
Waiter:
•	Selects available table
•	Creates new order
•	Adds items
•	Clicks "Send to Kitchen"
Database:
•	Order stored in orders.Orders
•	Items stored in orders.OrderItems
•	Kitchen entry created in kitchen.KitchenOrders
________________________________________
🟢 STEP 4: Kitchen Process
Kitchen Staff:
•	Sees new order
•	Marks items as Preparing
•	Marks items as Ready
System:
•	Updates order status
________________________________________
🟢 STEP 5: Order Completed
Waiter:
•	Serves food
•	Marks order as completed
________________________________________
🟢 STEP 6: Billing Process
Cashier:
•	Opens completed order
•	System auto-calculates:
o	Subtotal
o	Tax
o	Total
Cashier:
•	Select payment method
•	Confirms payment
Database:
•	Entry added in billing.Bills
•	Entry added in billing.Payments
________________________________________
🟢 STEP 7: Table Released
System:
•	Table status updated to Available
•	Ready for next customer
________________________________________





5️⃣ AUTOMATION & TRIGGERS (DATABASE LOGIC)
Your system uses triggers for:
🔁 Inventory Auto Deduction
When order is placed:
•	Inventory stock automatically reduces
🔄 Order Status Sync
When kitchen marks order ready:
•	Order status updates automatically
🧾 Bill Auto Calculation
When order completed:
•	Bill record auto-created
